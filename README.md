# Taskify

is an small app built with rails 7 + stimulus following this awesome tutorial https://www.akshaykhot.com/building-to-do-list-using-hotwire-and-stimulus/
## Installation


```bash
bin/bundle install
```

```bash
bin/rails db:create
```

```bash
bin/dev
```
